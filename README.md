# ClearCover
clearcover takehome


### Steps to launch application

1. clone repo
```
git clone https://github.com/Pnickolas1/ClearCover.git
```
2. cd into ClearCover dir
```
cd ClearCover
```
3. npm install
```
npm install
```
4. start application -> (Use http://0.0.0.0:3000)
```
npm run start
```
5. to test application
```
npm run test
```
